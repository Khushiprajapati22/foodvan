# 🍔 FoodVan - Online Food Ordering System

## 📌 Project Description

FoodVan is a web-based food ordering system where users can browse food items, add them to cart, and place orders easily.
It also includes admin functionalities to manage inventory, orders, and notifications.

---

## 🚀 Features

### 👤 User Side

* User Registration & Login
* Browse Food Items
* Add to Cart
* Place Orders
* View Cart Status

### 🛠️ Admin Side

* Manage Food Items
* Update Inventory Stock
* Order Management
* Notifications for Users & Admin

---

## 💻 Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: PHP
* Database: MySQL
* Server: XAMPP / Localhost

---

---

## ⚙️ Installation Steps

1. Clone the repository:

```
git clone https://github.com/Khushiprajapati22/foodvan.git
```

2. Move project to XAMPP htdocs folder

3. Start Apache & MySQL in XAMPP

4. Import database in phpMyAdmin

5. Open browser:

```
http://localhost/foodvan
```
